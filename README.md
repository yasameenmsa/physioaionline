# PhysioAI.online

AI-powered exam preparation platform for physiotherapy students.

## Quick Start

The main application code is in the `physioai/` directory.

```bash
cd physioai
npm install
npm run dev
```

See [physioai/README.md](./physioai/README.md) for detailed documentation.

## Documentation

Project documentation is available in the `specs/` directory using Spec Kit framework.

## Tech Stack

- Next.js 14, React 18, TypeScript
- MongoDB with Mongoose
- NextAuth.js v5 for authentication
- Tailwind CSS + shadcn/ui components
